## Installation

Install:

```
npm install --save @gltf-transform/core
```

Import:

```js
// ES6
import { Document, Scene, WebIO } from '@gltf-transform/core';

// CommonJS
const { Document, Scene, WebIO } = require('@gltf-transform/core');
```
